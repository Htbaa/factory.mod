htbaapub.factory

Factory Design Patter

The factory design pattern enables you to create all kinds of objects of different types with only using one factory object. When you're using some kind of scripting language inside your game, like Lua, all you need to do is expose your factory method to your scripting virtual machine. That saves an aweful lot of time when you'd normally have to expose all your constructors to your scripting virtual machine.
